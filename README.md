# WBO Animation - генератор видео-карточек

Десктопное приложение для Windows на Python, которое:

- накладывает видео (режим наложения «Экран») на пачку изображений (карточки маркетплейса);
- позволяет для каждой карточки выбрать своё видео и настройку «кривых»;
- экспортирует готовые видео-карточки в формате `.mp4`.

## Требования

- Windows 10/11;
- установленный Python 3.10+ (для разработки/сборки);

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

На Windows можно запускать двойным кликом по `run.bat` или командой:

```bash
python main.py
```

## Сборка в .exe (для запуска без Python)

Можно собрать один исполняемый файл для Windows с помощью PyInstaller:

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Запустите скрипт сборки:

   ```bash
   build_exe.bat
   ```

3. Готовый файл `WBOAnimation.exe` появится в папке `dist`.  
   Для корректной работы рядом с exe должны лежать папки `Assets/video` и `Assets/demo`.

## Структура ассетов

- `Assets/video` — сюда кладём видео для наложения (`.mp4`, до 8 секунд, 900x1200 px);
- `Assets/demo` — демонстрационные карточки (изображения), которые можно загрузить в приложение.
